# roots-ayx-guide

本仓库用于归档和发布多个独立 HTML 页面，作为页面资源的统一存放与展示空间。

## 项目简介

`roots-ayx-guide` 是一个静态 HTML 页面归档仓库。仓库内收录了若干独立制作的页面文件，这些页面不针对任何特定域名或网站，仅供浏览、参考或存档用途。

## 目录说明

```
roots-ayx-guide/
├── index.html          # 仓库首页 / 页面索引
├── pages/              # 归档页面存放目录
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放各独立 HTML 页面文件。
- `assets/` 目录包含页面引用的样式、脚本及图片资源。
- 首页 `index.html` 提供页面列表或导航入口。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开。
- 页面内容不涉及具体域名或商业推广，仅作为技术或内容存档。
- 仓库内所有页面均以静态方式呈现，无需后端支持。

## 维护说明

- 本仓库由维护者不定期更新，新增或更新页面将直接提交至对应目录。
- 欢迎通过 Issue 或 Pull Request 提交建议或修正。
- 请勿在页面中添加外部链接、广告或诱导点击内容。
- 页面内容应保持简洁、清晰，符合本仓库的归档定位。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意页面。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。

## 许可

本仓库内容仅供学习、参考与归档使用。如需引用或分发，请遵守相关法律法规并注明出处。
